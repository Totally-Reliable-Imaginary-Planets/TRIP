## Installation

Add this to your `Cargo.toml`:

### SSH
```toml
[dependencies]
trip = { git = "git@github.com/Totally-Reliable-Imaginary-Planets/TRIP.git", branch = "main" }
```

### HTTPS
```toml
[dependencies]
trip = { git = "https://github.com/Totally-Reliable-Imaginary-Planets/TRIP.git", branch = "main" }
```
